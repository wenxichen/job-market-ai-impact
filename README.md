# AI Job Market Predictions

A neutral archive of public predictions about how AI will affect jobs, employment, wages, occupations, and labor markets.

## Phase 1 goal
Collect predictions first.

No scoring, no commentary, no judgment.

## Inclusion rule
Include only public, sourceable predictions from notable people or institutions, such as:
- CEOs / founders
- politicians / regulators
- economists
- prominent AI researchers
- public intellectuals / commentators
- major institutions

## Minimum evidence standard
Each entry should have:
- exact quote or close paraphrase
- original source link
- speaker name
- date made
- at least one topic tag
- at least one time-horizon field when possible

## Suggested categories
- job displacement
- job creation
- wages
- productivity
- automation exposure
- white-collar work
- blue-collar work
- software engineering
- creative work
- education / retraining
- AGI / superintelligence implications
- regulation / policy

## Neutrality rule
Do not add opinions to entries.
Do not classify a prediction as true/false in phase 1.
Do not rewrite strong claims into weaker ones.
Preserve the original meaning as faithfully as possible.

## Website files
The first working site lives in:
- `index.html` — archive homepage with search and filters
- `prediction.html` — prediction detail page
- `styles.css` — shared styles
- `app.js` — homepage logic
- `prediction-detail.js` — detail page logic

## Local preview
From this folder, run:

```bash
python3 -m http.server 4173
```

Then open:
- `http://127.0.0.1:4173/index.html`
- `http://127.0.0.1:4173/prediction.html?id=sam-altman-2024-01-18-wef-001`
